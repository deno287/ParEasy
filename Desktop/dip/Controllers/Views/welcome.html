ParEasy

making GPU parallel programming easy


Powerful graphics card

This is a web application running on a server with a powerful graphics card with a graphics coprocessor for parallel computing (GPU).

Running projects

The system is an developed for creating, debuging and running programs written in a simple educational programming language.

Educational language

The language has a built-in support for efficient parallel computation, utilizing the computing power of GPU including SIMD parallelism.

Beautiful examples

Student can see several examples with the new language in Lectures section, where the ways of parallel programming is clearly illustrated.
